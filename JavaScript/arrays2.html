Array Problem Set

Problem set.
  1. Write a function printReverse() that takes an array as an argument and prints the elements out to the console in reverse order.

  2. Write a function isUniform() which takes an array as an argument and returns true if all its elements are identical.

  3. Write a function sumArray() that accepts an array of numbers and returns their total combined sum.

  4. Write a function max() that accepts an array of numbers and returns the highest number in it.
